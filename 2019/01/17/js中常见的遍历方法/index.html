5.map方法

array.map(callback,[ thisObject]);

执行结果：

说明：
返回值：返回一个由原数组中的每个元素调用一个指定方法后的返回值组成的新数组。
是否改变原数组：不改变原数组
常用场景：常用于对数组的每一项做相同的操作并返回一个新的数组

6.filter方法

执行结果：

说明：
返回值：返回一个由原数组中的每个元素调用过滤方法后的返回值组成的新数组。
是否改变原数组：不改变原数组
常用场景：常用于在数组中查找符合条件的项，并返回一个新数组

7.some方法

执行结果：

说明：
返回值：为数组中的每一个元素执行一次 callback 函数，直到找到一个使得 callback 返回一个“真值”（即可转换为布尔值 true 的值）。如果找到了这样一个值，some 将会立即返回 true。
注意：some如果遇到 true 就不再执行了。
是否改变原数组：不改变原数组
常用场景：常用于在数组中查找符合条件的唯一项，例如根据id查找某一项，如果 return true，就会立即终止这个数组的后续循环
